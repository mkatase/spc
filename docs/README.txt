#-----------------------------------------------------------------------
@en
## Overview
A lightweight, text-based drawing tool for empirical shape rendering using an intuitive coordinate system.

## About
Survival Painter (SPC) is designed to render shapes like circles and rectangles using minimal code and concise commands.
It is a drawing tool for developers who prefer staying in their text editor to construct quick diagrams or pixel-perfect logic without launching heavy GUI applications.

## Environment
@end
@jp
## 概要
テキストベースの直感的な座標系を用いて、図形を「実証的」に描画する軽量ツール

## About
Survival Painter (SPC) は、円や矩形といった図形を、限りなく少ない記述量と簡潔な命令のみで描画できるように設計されています。
GUIツールを立ち上げるまでもない「ちょっとした図案」を、テキストエディタだけで構築するための、エンジニア向けのドローイングツールです。

## Features
- Origin is at bottom-left
- Concise and minimal syntax

## Careful Points
- Drawing procedure operations are processed sequentially from the top to the bottom of the file (Sequential Overwriting).

@end
@jp
## 特徴
− 原点は左下
- 簡潔で少ない記述量

## 注意すべき点
- 描画は、ファイルの上から下へ順に処理(上書き)されます。

## Execution Mode
| Mode  | Description                        |
|:------|:-----------------------------------| 
| check | Syntax check of input data         |
| view  | Instant preview display            |
| run   | Output image files (webp/png)      |
| image | Low-load output (Some limitations) |
| svg   | Vector output (SVG only)           |

@end
@jp
## 実行モード
| モード| 説明                            |
|:------|:--------------------------------|
| check | データの構文チェック            |
| view  | 画像のプレビュー表示            |
| run   | 画像の出力(webp/png)            |
| image | 画像の出力(低負荷/一部制限あり) |
| svg   | SVGファイルの出力               |
