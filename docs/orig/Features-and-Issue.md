## SPCの大きな特徴
- 原点が左下
- ワンライナー記述

コンピュータの原点は左上だが、人の意識としての原点は左下だと思われる。直感的な描画速度を向上させるために、左下を原点としている。
また、一部のタグは異なるが、単純な図形を描画させるのに対して、ワンライナー記述を行えば、描画または画像出力ができることを特徴としている。

## 制限点
- AR(円弧）がimageモードにおいて改善が必要
- GR(グルーピング)やRO(回転)でのネストは不可
- テキストの回転はsvgモード限定
- TE（テキスト）用フォントが固定
